## 遇到的错误

| 错误 | 尝试次数 | 解决方案 |
|------|---------|---------|
| Maven 编译: 不支持发行版本 5 | 1 | pom.xml 添加显式 maven-compiler-plugin release=17 |
| H2 SQL: AUTO_INCREMENT 语法不兼容 | 1 | 改用 GENERATED BY DEFAULT AS IDENTITY |
| PostgreSQL 连接: no password provided | 1 | Demo 改用 H2 内存库默认；云数据库连接单独配置 |

## 技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 密码哈希 | BCrypt | Spring Security 标准，与 exam_web 一致 |
| JWT 库 | jjwt 0.12.5 | pom.xml 已引入 |
| 认证方式 | JWT Bearer Token | API 接口规范要求，前后端分离 |
| 权限模型 | sys_uac_url 接口级 | 与 exam_web 一致，js=9 超级管理员 |
